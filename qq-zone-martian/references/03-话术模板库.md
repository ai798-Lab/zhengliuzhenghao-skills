# 葬爱话术模板库

> 那个年代的非主流话术不是随便写的，是一套完整的修辞体系。
> 主打"明明没经历过什么，却像看破红尘"的疏离感。

---

## 一、45° 仰望天空系列（核心句式）

这是非主流文学的祖师爷。结构：**动作 + 角度 + 哲思**

```
我以45°角仰望天空，眼泪才不会掉下来。
当我以45°角仰望天空时，我才发现，原来寂寞也是一种美。
有人说，仰起头45°，眼泪就不会流下来。我试了，结果，淹死了。
45°的天空，是属于我一个人的悲伤。
仰望45°的天空，那里有属于我的回忆。
我用45°的角度仰望天空，把眼泪逼回去。
```

**变体公式**：`[我/某人] + 用45°的角度 + [仰望/凝望/眺望] + [天空/星空/夕阳] + [，] + [感叹句]`

## 二、伤痛文学（葬爱主线）

### 套路 1：用情感对立营造矛盾感
```
明明笑着，却感觉眼泪要流下来。
明明很在乎，却装作不在乎。
明明很爱你，却不能在一起。
明明说好不哭，可是眼泪就是不听话。
```

### 套路 2：万物皆可"忧伤的"
```
忧伤的旋律
忧伤的青春
忧伤的城市
忧伤的雨
忧伤的背影
忧伤的烟花
那一抹忧伤，划过我的脸颊。
```

### 套路 3：把"我"换成"那个女孩 / 那个男孩"
```
那个女孩，习惯了一个人。
那个男孩，曾经也很温柔。
那个女孩，把眼泪藏在笑容里。
那个曾经爱笑的女孩，再也不笑了。
```

### 套路 4：宿命感
```
我的世界，从此与你无关。
有些人，注定只能错过。
故事的开始总是极好的，结局却各自天涯。
你是我猜不到的不知所措。
擦肩而过，是最遥远的距离。
```

## 八、写作风格要点

1. **主语爱用 "莪"，宾语爱用 "沵"**
2. **每句末尾加句号、空格、或符号**（不要用现代标点节奏）
3. **频繁用"ヽ" 代替逗号**
4. **关键词用空格隔开**：`莪 dē 訫 . 詠遠 . 噯 沵`
5. **必出现的高频词**：寂寞、回忆、忧伤、永远、青春、放不下、回不去、错过、缘分
6. **绝对不能出现的词**：内卷、躺平、绝绝子、yyds、栓Q（这些都是后来的梗，年代不对）
