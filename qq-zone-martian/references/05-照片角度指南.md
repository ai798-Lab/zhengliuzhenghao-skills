# 45° 仰望天空照片角度指南

> 非主流自拍的灵魂——不是你的脸有多好看，而是角度有多到位。
> 这套指南覆盖：自拍角度、场景选择、滤镜参数、配文模板。

---

## 一、传说中的 45°（数学派）

**角度定义**：手机 / 相机镜头与水平面成 45° 夹角，向上仰拍。

**作用**：
1. 显得脸小（透视压缩下颌）
2. 眼睛显大（眼球位置上抬）
3. 显得忧郁（眉骨投影到眼眶）
4. 天空入镜（自动获得"哲思感"背景）

**禁忌**：
- ❌ 不能正脸——太阳穴
- ❌ 不能 90° 完全朝天——拍到鼻孔
- ❌ 不能逆光过曝——细节全无

## 二、经典 Pose 八连

### Pose 1：单手撑下巴 + 望远方
```
✦ 角度：45°仰视
✦ 手势：右手托腮，食指放在嘴角
✦ 视线：望向画面外左上方
✦ 表情：嘴角微扬但眼神空洞
```

### Pose 2：手扶刘海 + 望天空
```
✦ 角度：50°仰视
✦ 手势：左手轻轻撩起额前刘海
✦ 视线：闭眼或望天
✦ 表情：嘴唇微张
```

### Pose 3：耳机线入镜 + 侧脸
```
✦ 角度：45°侧仰
✦ 道具：白色耳机线必须入镜
✦ 视线：望向窗外
✦ 表情：若有所思
```

### Pose 4：嘟嘴比 V
```
✦ 角度：30°仰视
✦ 手势：右手在脸侧比 V，靠近眼睛
✦ 表情：嘟嘴卖萌
✦ 备注：可爱型非主流标配
```

### Pose 5：捂半边脸
```
✦ 角度：45°
✦ 手势：用刘海或手挡住半张脸（通常挡右脸）
✦ 视线：仅露出的那只眼朝下
✦ 表情：神秘忧郁
```

### Pose 6：低头玩手机自拍
```
✦ 角度：手机举高俯拍自己玩手机的动作
✦ 视线：盯着手机屏幕
✦ 表情：嘴角抿
✦ 文案：「莪dē世界呮宥這個小屏幕」
```

### Pose 7：靠墙 + 头微仰
```
✦ 场景：靠涂鸦墙 / 红砖墙
✦ 角度：45°仰拍
✦ 表情：闭眼
✦ 配文：「靠著墙ヽ听著歌ヽ莪dē世界沵卟懂」
```

### Pose 8：玻璃窗倒影自拍
```
✦ 场景：雨天的玻璃窗 / 公交车窗
✦ 拍法：拍窗外 + 倒影里的自己
✦ 文案必带"雨"字
```

## 四、滤镜参数（美图秀秀年代）

那个时代不用 VSCO，用美图秀秀。经典参数：

### 「LOMO 风」（最常用）
```
亮度：+15
对比：+30
饱和：-20
暗角：开启
颗粒：轻度
```

### 「樱花粉」
```
色温：偏冷
柔焦：开启
亮度：+20
肌肤：磨皮 +50
眼睛：放大 +30
```

### 「黑白忧郁」
```
模式：去色
对比：+40
暗角：强
锐度：+10
```

### 「过曝小清新」
```
亮度：+40
高光：+30
饱和：-30
柔焦：开启
```

### 必装效果
- ✦ 大眼瘦脸 +50
- ✦ 磨皮（必须脸看不见毛孔）
- ✦ 肤色提白
- ✦ 边框：彩虹 / 蕾丝 / 心形
- ✦ 闪图（动态贴纸）
- ✦ 文字水印（火星文签名）

## 五、配文模板（照片说明）

### 单图说明
```
场景：[地点]
心情：[情绪火星文]
BGM：[那个年代的歌]

[一句45°仰望式哲思]
                                ——丶葬愛灬[名字]
```

### 九宫格主题
- **「莪dē青春纪念册」**：从小到大的照片混排
- **「卋界很大ヽ莪很小」**：风景 + 自拍
- **「沵卟在dē日子」**：空椅子、空房间、单人自拍
- **「最後dē微笑」**：黑白 + 嘴角上扬

## 七、出片建议（输出时给用户的建议）

当用户说"我想发空间动态"或"我要拍照"时，按这个套路给建议：

1. **选场景**：从上面 Top 10 挑一个匹配心情的
2. **定 Pose**：从八连里推荐 1-2 个
3. **滤镜**：根据场景选预设
4. **配文**：从话术模板库 + 火星文转换 生成
5. **BGM**：从时代文化元素库挑一首
6. **可见范围**：必须强调「仅 QQ 好友可见」或「指定分组可见」（神秘感）
7. **发布时间**：午夜 23:00-02:00 之间（"睡不着才发的"）
