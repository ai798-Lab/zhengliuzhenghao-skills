## 七、英文 + 数字 + 拼音夹杂规则

火星文经典手法是中文 + 拼音字母 + 数字混排：

- **拼音替代**：的 → `dē`、爱 → `āi`、心 → `xīn`
- **声调拼音**：莪dē訫、沵dē噯
- **数字谐音**：520(我爱你) / 1314(一生一世) / 7456(气死我了) / 88(拜拜) / 9494(就是就是)
- **字母装饰**：v、k、ぃ、っ 插在中间

**经典示范**：
- 普通：我的心永远爱你
- 火星：莪dē訫詠遠噯沵
- 进阶：莪 dē 訫`★`詠遠`丶`噯沵 ωǒ

## 八、繁体强制转换（伪文艺加成）

非主流的另一招：能用繁体绝不用简体。

- 爱→愛 / 国→國 / 学→學 / 听→聽 / 这→這 / 时→時 / 来→來 / 后→後 / 万→萬 / 离→離 / 远→遠 / 梦→夢 / 风→風 / 没→沒 / 头→頭 / 们→們 / 个→個 / 会→會 / 长→長 / 关→關 / 间→間

## 九、转换强度等级

根据用户需求选择强度：

- **🌶 轻度**（30% 替换）：保持可读，只换核心词。适合"想看个意思"
- **🌶🌶 中度**（60% 替换）：明显火星化，但能读懂。**默认推荐**
- **🌶🌶🌶 重度**（90% 替换）：基本全换，需要稍微辨认。适合纯整活
- **🌶🌶🌶🌶 究极**：火星文 + 符号 + 拼音 + 繁体 + 全角 全部叠加。适合做静态展示截图

## 十、转换示范

**原文**：今天天气很好，我和朋友去公园玩，很开心。

| 强度 | 输出 |

| 🌶🌶 | 妗兲兲气狠恏，莪咊萠囿厾公园玩，狠開吢。 |
| 🌶🌶🌶 | 妗兲兲気哏恏丶莪咊萠囿厾叴園頑丶哏開吢╰ |
