# 符号装饰库

> 火星文只是基础。真正的非主流灵魂，在符号装饰。
> 名字前后缀、句末小尾巴、心情符号、分隔符——一个都不能少。

---

## 一、网名前后缀（葬爱家族标配）

### 前缀类
```
丶  灬  ╰  ╮  ╭  ㄣ  ぶ  の  っ  ぃ  ▍  ▎  ◆  ◇  ↘  ↗
```

### 后缀类
```
_、  ︶ㄣ  丶  ↘  ★  ☆  ♀  ♂  ヾ  づ  oО  oo  -  ▼
```

### 经典网名结构
```
[前缀] + [核心字] + [小尾巴]
```

**示范**：
- 灬 葬愛丨丿
- ╰つ夜未眠
- 丶薆絔誐
- ╮殇丨小毅
- ㄣ流年丿不悔
- づ末日下dē那抹忧伤
- ◆◇╮丶冷丨無心
- 丿丶葬の翼

### 葬爱家族经典姓氏
```
葬  殇  瞳  毅  痕  夜  烟  泪  魇  魂  寒  暮  陌  墨  幽  孤  雪  冷
```

( ⊙o⊙ )   ╮(╯3╰)╭   (>_<)   T_T   QAQ   QwQ

## 三、装饰性符号串（句中分隔）

```
丶  ㄝ  ヾ  *  ★  ☆  ♀  ♂  ◆  ◇  ●  ○  ▲  △  ◢  ◣  ◤  ◥
↗  ↘  ↖  ↙  ↑  ↓  ←  →  ↔  ▼  ▲  ◀  ▶
※  ❀  ✿  ❁  ❃  ✾  ✽  ❅  ❆  ☃  ❄
♪  ♫  ♬  ♭  ♮  ♯  ♩  ♨
```

**用法**：火星文每隔几个字插一个，营造"我很在乎排版"的错觉。

```
莪 ★ 詠遠 ☆ 噯 ♀ 沵
```

## 四、彩虹字描边模板（文字描述用）

那个年代的真彩虹字需要在 QQ 空间里用代码生成，文本里通常用这些方式暗示：

```
[ft=#FF0000,,]红字[/ft]
[ft=#00FF00,,]绿字[/ft]
[B]加粗[/B]
[I]斜体[/I]
[fly]飞行字[/fly]
```

如果是在普通文本里模拟，可以用方括号包围 + 描述：
```
【❤红色】莪dē訫【❤红色】詠遠噯沵
```
