## 🎯 这是什么

一个 [Claude Code](https://claude.com/claude-code) 的 [Agent Skill](https://docs.claude.com/en/docs/claude-code/skills)。它不是简单的火星文字符替换器，而是一整个 **2010 年 QQ 空间宇宙的考据复原工程**。

包含：

- ✦ **200+ 火星文字符映射**（生僻字 / 假名 / 注音 / 繁体）
- ✦ **完整的葬爱话术体系**（伤痛 / 装酷 / 海誓山盟 / 45°仰望）
- ✦ **2008-2012 文化考据库**（许嵩、徐良、QQ 钻、明晓溪小说、农场偷菜）
- ✦ **杀马特视觉美学复原**（蕾丝 / 闪图 / 彩虹字 / 模块化 QQ 空间布局）
- ✦ **HTML 渲染管线**（用户输入 → AI 生成 HTML → Puppeteer 截图 → PNG）

```bash

```

## 💡 使用

进入任何配好这个 skill 的目录，启动 Claude Code：

```bash
claude
```

然后输入：
```
> 给我做一个非主流宇宙：我叫张三，是大学生，喜欢打篮球
```

Claude 会自动：
1. 解析你的身份特征
2. 生成葬爱姓名 + 家族 + 人设
3. 写一篇符合你身份的火星文日志
4. 渲染成 HTML + PNG
5. 把文件路径告诉你

输出文件：
- `universe/张三.html` — 可在浏览器打开
- `universe/张三.png` — 长截图，可直接发小红书

## 🎨 设计理念

**非主流宇宙生成器** 的本质是 **"给每个人一封写给 16 岁自己的情书"**。

那个时代的 QQ 空间承载着我们最矫情、最真实、最不堪回首的青春。这个 skill 不是为了嘲讽——它是用 2026 年最先进的 AI，认真严肃地复原一个我们曾经真实存在过的精神角落。

你输入的每一个字，都会被翻译成那个年代的语言；
你的每一个身份特征，都会被嵌入葬爱家族的话术体系；
你的现代生活，会变成一篇 2010 年凌晨 3 点写下的火星文日志。
