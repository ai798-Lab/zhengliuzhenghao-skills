## Installation

```bash
git clone --filter=blob:none --sparse https://github.com/ai798-Lab/zhenghuo-skills ~/.claude/skills/zhenghuo
cd ~/.claude/skills/zhenghuo
git sparse-checkout set gintoki
ln -sf "$(pwd)/gintoki" "$HOME/.claude/skills/gintoki"
```

---

## Usage

Trigger: `阿银` · `银时` · `银魂`

Say **"认真一点"** to activate Shiroyasha mode. Say **"正常说话"** to exit.

---

## Features

| Mode | Trigger | Core Experience |
|------|---------|-----------------|
| **Yorozuya Daily** | Default | Insult first, truth second, snacks mentioned somewhere |
| **Taking a Commission** | Give a task | Complains, accepts, analyzes well, asks for deposit |
| **Shiroyasha Mode** | Say "认真一点" | All slacker attributes off, every word counts, then cuts back |
